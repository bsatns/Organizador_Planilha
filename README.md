**Divisor de Planilhas Excel**

Sistema desenvolvido em Python com Flask para automatizar a divisão de planilhas Excel entre colaboradores, facilitando a distribuição de atividades e reduzindo o trabalho manual.

**Funcionalidades**

- Upload de arquivos Excel (.xlsx)

- Seleção de colaboradores por setor

- Distribuição automática dos registros

- Geração de múltiplas planilhas individuais

- Opção de selecionar todos os colaboradores

- Interface web simples e intuitiva

- Download das planilhas geradas

- Adição dinâmica de novos colaboradores

**Tecnologias Utilizadas**

- Python

- Flask

- Pandas

- OpenPyXL

- HTML5

- Bootstrap

- JavaScript

**Como Funciona**

- Faça o upload da planilha Excel.

- Escolha os colaboradores que irão receber as atividades.

O sistema divide automaticamente os registros.
As planilhas individuais são geradas.
Faça o download dos arquivos prontos para utilização.
